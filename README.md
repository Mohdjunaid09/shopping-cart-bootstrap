# 🛒 Shopping Cart Application (Bootstrap)

## 📌 Project Overview

This project is a **responsive Shopping Cart web application** built using **HTML, CSS, JavaScript, and Bootstrap**.
It demonstrates how to create a modern UI with Bootstrap components and implement basic shopping cart functionality.

---

## 🚀 Features

* 🧾 Product listing with cards
* 🛍️ Add to cart functionality
* ➕ Increase / ➖ decrease quantity
* ❌ Remove items from cart
* 💰 Total price calculation
* 📱 Fully responsive design using Bootstrap
* 🧭 Navigation bar for easy access

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

## ⚙️ How to Run the Project

### Step 1: Clone the Repository

```bash
git clone https://github.com/Mohdjunaid09/shopping-cart-bootstrap.git
```

### Step 2: Open in VS Code

* Open the folder in **VS Code**
* Install **Live Server extension** (optional but recommended)

### Step 3: Run the Project

* Right-click on `index.html`
* Click **"Open with Live Server"**
